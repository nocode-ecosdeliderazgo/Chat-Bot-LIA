Diagnóstico de Adopción de IA – Compilado de Cuestionarios, Diccionario y Formularios
Documento maestro que integra todos los perfiles (Direcciones, Miembros, Consultores, Freelancer), un diccionario de respuestas unificado y formularios de captura por perfil.
Versión: 2.0
Diccionario de Respuestas (unificado)
Tipos de respuesta
•	Likert 1–7 (marcar un valor):
•	1 = Nulo / No lo conozco / Nunca lo uso
•	2 = Conozco el término, no lo aplico / Pruebas mínimas
•	3 = He probado poco / Uso ocasional y básico
•	4 = Uso regular en algunos casos / Aplicación inicial
•	5 = Uso frecuente / Impacta varias tareas
•	6 = Uso alto / La IA optimiza gran parte de mi trabajo
•	7 = Experto con experiencia práctica / Uso avanzado y estratégico
•	Multi-Selección: marque todas las opciones que apliquen.
•	Sí/No: Sí = 100, No = 0 (para cálculo).
•	Abierta: texto libre. (Opcional: bono cualitativo interno 0–2 puntos por ejemplo concreto y resultado).
Guía de interpretación (para devoluciones)
•	Índice 0–39 = Básico → Priorizar 2 casos simples y formación esencial.
•	Índice 40–69 = Intermedio → Escalar 2–3 flujos, estandarizar y medir.
•	Índice 70–100 = Avanzado → Automatizar, medir valor y difundir mejores prácticas.
Buenas prácticas al responder
•	Responda solo el cuestionario asignado por su perfil.
•	Use ejemplos reales en preguntas abiertas (si puede, añada métricas: % tiempo ahorrado, errores reducidos, etc.).
•	En multi-selección, seleccione todo lo que aplique.

Proceso de Evaluación de Resultados (unificado)
0) Estructura base (metadatos de cada pregunta)

Crea una tabla “Catálogo de Preguntas” (una fila por pregunta) con:

Perfil (CEO, Dir. Ventas, Miembros Marketing, etc.)

ID (p.ej. VENT-2.3)

Tipo = {Likert_1_7, MultiCheck, SíNo, Abierta}

Dimensión = {Conocimiento, Aplicación, Estrategia, Inversión, Productividad}

Subdominio (ej.: Ventas→Prospección / Pipeline / Reuniones; Compras→Sourcing / RFx / Proveedores / Órdenes / Gasto; Contabilidad→Captura / Conciliación / Cierre / SAT; RRHH→Reclutamiento / Capacitación / Desempeño; etc.)

Máximo lógico (para MultiCheck)

Esto permite reusar el mismo motor con todos los perfiles, solo cambiando metadatos.

1) Puntuación por tipo de pregunta (normalización 0–100)

A. Likert 1–7
Puntaje = ((Respuesta - 1) / 6) * 100

B. MultiCheck (marque todo lo que aplique)
Puntaje = (#seleccionadas / Máximo lógico) * 100
(Si no hay máximo fijo, usa el total de opciones.)

C. Sí/No
Sí = 100, No = 0.

D. Abierta (opcional, bono cualitativo)

0 = Vago / sin ejemplo concreto

1 = Ejemplo básico (describe uso)

2 = Ejemplo avanzado (describe uso + resultado/indicador)

BonoCualitativo = (Suma de bonificaciones / Máx. posible) * 10 (límite +10 puntos al total del perfil).

Si no deseas bonos, marca todas las abiertas como “No puntúan”.

2) Agregación por Dimensión y Subdominio

Dimensiones estándar (todas: directivos, miembros, consultores, freelancers):

Conocimiento (items de nivel general + familiaridad)

Aplicación (items de uso en procesos del área)

Productividad (si existe sección específica)

Estrategia (claridad de potencial / roadmap personal)

Inversión (disposición a capacitarse 12 meses)

Cálculo
Para cada Dimensión D:

Puntaje_D = PROMEDIO( Puntajes de preguntas con Dimensión = D )


Para cada Subdominio (ej. en Compras: Sourcing, RFx, Proveedores, Órdenes, Spend):

Puntaje_Sub = PROMEDIO( Puntajes vinculados al Subdominio )


Si una persona no ve preguntas de un subdominio (por el flujo), ese subdominio se marca N/A y no afecta el promedio.

Índice = Σ (Peso_Dimensión * Puntaje_Dimensión) + BonoCualitativo (máx +10)

5) Reglas de “banderas” y oportunidades

Genera señales automáticas para el reporte:

Bandera de Capacitación: Conocimiento < 50 y Inversión ≥ 60
→ “Alto apetito de aprendizaje con brecha de base: priorizar curso esencial + práctica guiada.”

Bandera de Adopción: Aplicación < 50 y Conocimiento ≥ 60
→ “Sabe, pero aún no aplica: activar 2 casos de uso concretos + coaching 4 semanas.”

Bandera de Eficiencia: Productividad < 50
→ “Potencial para ahorro de tiempo: implementar resúmenes automáticos, plantillas, y automatización de tareas repetitivas.”

Bandera de Patrocinio (para Directores): Estrategia < 50
→ “Falta de dirección: definir roadmap personal de 90 días con 3 hitos e indicadores.”

Riesgo de Declarativo: Conocimiento ≥ 70 y Aplicación ≤ 40
→ “Diferencia teoría–práctica: diseñar piloto de 2 semanas, medir y escalar.”
